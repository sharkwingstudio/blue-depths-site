Changelog

IMPORTANT [Exclusively for players playing 26.1-26.1.2]

This current version 100% works for 26.1 to 26.1.2; please ignore all other versions for 26.1 to 26.1.2. Apologies to those who tried to install the datapack in 26.1 and above, previously. When initially testing out the datapack, the test was done on a pre-26.1 world, because of this I was not aware that the pack did not work when installed on a brand-new world. If possible I will delete those other releases to prevent any issues.

News

This is the final patch (unless I receive feedback on any major bugs) for the initial release of Blue Depths. The next update will focus more on adding new, exciting content for Blue Depths. Along with bug fixes, 1.0.3 lays the groundwork for the upcoming new content. The next two major content updates will revolve around the same theme, expanding rivers to make them feel more alive. Keep an eye out for the next few weeks for the first release of the update, Roaring Rivers.

Bug Fixes & Technical Changes

  • Fixed a bug where players in creative or spectator mode would receive the sting effect when near the box jellies
  • The mobs (aside from the catfish) no longer disappear when you use a bucket on them
  • Laid the groundwork for a major mechanic that will be released in the upcoming Roaring Rivers update.
