Guide Book

Use

The Guide Book is a useful guide into the basics of the Blue Depths datapack. It offers crafting recipes for all the custom items and blocks in the pack, gives a brief description for each mob, explains life cycles, and gives information on how to summon the Nameless Depth boss. In most pages where crafting is involved, there are images available to help explain how the crafting/cooking pattern should look like.

Obtaining and Crafting the Guide Book

As of version 1.0.1, all new players will spawn with one copy of the guide book upon log-in. For every major update, the player will always receive one copy of an updated version of the guide book on log-in that contains new information on the new mobs, items, and blocks added by the update, or it will contain updated information on a system if it was changed (like crafting recipes, breeding instructions, etc).

If for some reason the guide book is lost or the player would like to obtain another copy for the guide book for whatever reason, the guide book can be crafted using the inventory crafting grid of a crafting table. All the player has to do is place a book and salmon inside the crafting grid to craft the book. Order does not matter.
